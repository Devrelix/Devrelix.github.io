Privacy Policy

This Privacy Policy describes how GPTPrint Pro – Print & Export ChatGPT to PDF ("the Extension") operates.
Last Updated: March 2026

1. Introduction

Devrelix (“we”, “our”, or “us”) respects your privacy.

GPTPrint Pro is designed to provide advanced printing customization for ChatGPT conversations while maintaining strict privacy standards. We do not collect or process personal data beyond what is necessary for the extension to function locally in your browser.

2. Data Collection

GPTPrint Pro:
  • Does not collect personal information
  • Does not store ChatGPT conversations
  • Does not transmit user data to external servers
  • Does not track browsing activity
  • Does not use analytics or advertising trackers
  • Does not sell or share user data

All extension functionality operates entirely within your browser.

3. Local Storage Usage

The extension uses Chrome’s storage API to store:

  • Theme preferences
  • Print customization settings
  • UI configuration options

This data:

  • Is stored locally in your browser
  • May sync across your devices if Chrome Sync is enabled
  • Is never transmitted to Devrelix or third-party servers

4. Permissions Explanation

GPTPrint Pro requires the following permission:

storage
Used to save user preferences and customization settings locally, such as theme selection, layout configuration, watermark settings, and header customization.

The extension also runs on the following websites:

  • https://chatgpt.com/*
  • https://chat.openai.com/*

These host permissions allow the extension to modify the print layout of ChatGPT conversations directly within the page so users can generate clean PDF exports.

The extension does not access unrelated browsing data and does not operate on any other websites.

5. Third-Party Links

The extension may include links to external platforms such as:

  • GitHub
  • Social media platforms
  • Support pages

We are not responsible for the privacy practices of third-party websites. Users are encouraged to review their respective privacy policies.

6. Data Security

Because GPTPrint Pro does not collect or transmit personal data to external servers, there is minimal risk of data exposure through our systems.

All processing occurs locally within the user’s browser environment.

7. Children’s Privacy

GPTPrint Pro is not directed toward children under the age of 13. We do not knowingly collect personal data from children.

8. Changes to This Policy

We may update this Privacy Policy as the extension evolves. Updates will be reflected with a revised “Last Updated” date at the top of this document.
